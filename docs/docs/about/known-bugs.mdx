Known bugs as well as enhancement ideas are tracked using GitHub issues. Please report any bugs or enhancements here being sure to provide a full explanation, preferably with images or better still, videos of the issue.

## Questions and Discussion Boards

We maintain discussion boards in GitHub for the different versions of RAVE. If you have questions or have requests for new features, please post them to the appropriate board:

- [QViewer Discussions & Questions](https://github.com/Riverscapes/QRAVEPlugin/discussions/categories/q-a)
- [ArcPro Viewer Discussions & Questions](https://github.com/Riverscapes/ArcProViewer/discussions/landing)
- [WebViewer Discussions & Questions](https://github.com/Riverscapes/rs-web-monorepo/discussions) (Private repo)

## Bugs

Before logging a suspected bug, please search the bugs for the relevant bugs below. If you have more information to report about this bug, please post a comment to the appropriate issue. 

- [QViewer Bugs](https://github.com/Riverscapes/RaveAddIn/issues?q=is%3Aopen%20is%3Aissue%20label%3A%22%3Abug%3A%20bug%22)
- [ArcPro Bugs](https://github.com/Riverscapes/ArcProViewer/issues)
- [WebViewer Bugs](https://github.com/Riverscapes/rs-web-monorepo/issues) (This repo is private. Public feedback can be provided to support@riverscapes.freshdesk.com)
