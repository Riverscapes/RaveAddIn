# Minimum Requirements

RAVE requires ArcGIS 10.4 or newer. RAVE does **not** work with ArcPro.

## ArcGIS Add-In

To install the RAVE Add-In for first time to ArcGIS 10.4 or later, simply double-click on [downloaded](https://github.com/Riverscapes/RaveAddIn/releases/latest) `*.ESRIAddin` file, while **NOT** in an active ArcMap sesssion.
