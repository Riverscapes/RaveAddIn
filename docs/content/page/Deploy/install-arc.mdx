To install the Riverscapes Viewer for first time in ArcGIS 10.x:

1. Close all ESRI products.
1. Click to [download](https://github.com/Riverscapes/RaveAddIn/releases/latest) the latest Riverscapes Viewer addin file.
1. Once the file is downloaded, double click it to install the AddIn. You will be prompted with a confirmation that the installation is complete.
1. Restart ArcGIS and the Riverscapes Viewer toolbar will be available as one of the top level Menu items.
